Brightwater Sensing

The purpose of the project is to process satellite imagery from multiple sources, combine it with data and visualize to get the near real-time estimates of crop yield.

I was in charge of creating the architecture and development of the Web-based UI, API endpoints and connecting them together.

The end product includes a geographical map with multiple layers, filters and time-based background synchronisation process.

Technologies used:

  • TypeScript
  • NodeJS
  • Postgres
  • Webpack
  • React&Redux
  • WebSockets
  • Mapbox
  • D3.js
  • Docker
  • nginx
  • AWS
